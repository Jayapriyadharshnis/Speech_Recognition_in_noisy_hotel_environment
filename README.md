# Speech Recognition in Noisy Hotel Environment using MATLAB with GUI

## Overview

This project implements a speech recognition system using MATLAB with a graphical user interface (GUI) to display the output. The system is designed to recognize speech accurately even in noisy environments such as hotel receptions, lobbies, and public areas.

The GUI allows the user to load or record speech and view the recognition result visually, making the system more user-friendly and interactive.

---

## Objective

To develop a MATLAB-based speech recognition system with GUI that can recognize spoken words reliably in noisy environments.

---

## System Workflow

1. The user loads or records a speech signal using the GUI  
2. The speech signal is preprocessed to reduce noise  
3. Important speech features are extracted using MFCC or PNCC  
4. The extracted features are compared with stored reference data  
5. The system recognizes the spoken word  
6. The recognized output is displayed in the GUI
